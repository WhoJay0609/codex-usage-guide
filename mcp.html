Concept

MCP / Plugins

MCP 和 plugins 让 Codex 连接外部工具，例如 GitHub、设计工具、数据库、文档库和搜索服务。

本页它是什么什么时候用可复制示例

What

它是什么

MCP 是工具接口层，plugins 是打包后的能力组合。它们让 Codex 不只读本地文件，还能操作外部系统。

When

什么时候用 / 不用

需要实时外部状态、设计同步、GitHub 操作、数据库查询时使用。涉及 secret、生产数据或写操作时，必须先确认权限边界。
